## Workflow 1: Full Launch Asset Set from a Single Positioning Doc

**Problem:** Product launches require the same information reformatted 6+ times — one-pager, landing page, email, ad copy, social posts, BD talk track. Doing this manually takes days and introduces inconsistency.

**Solution:** One positioning source of truth → Claude generates the full asset set in a single pass. I review, edit the strategic inputs, and ship.

### Input
A positioning doc with:
- Product name and one-line description
- Target ICP (who buys, what role, what company size)
- Core problem it solves
- Key differentiators (3 max)
- Proof points / metrics
- Call to action
